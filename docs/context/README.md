# SERAYA Context

This folder contains the small set of shared project notes that are useful for Kah Yan and Cheryl to exchange through GitHub during active development.

## Shared Files

- `product-brief.md`: stable product goals, users, MVP scope, design principles, and out-of-scope items.
- `open-questions.md`: unresolved questions to discuss with Cheryl.
- `decisions.md`: confirmed decisions, dated and concise.
- `mvp-question-set.md`: first proposed senior-facing MVP check-in questions.
- `mvp-resource-set.md`: first proposed MVP resource/link set.

## Local Files

Personal notes and repo cheatsheets should use the `.local.md` suffix, for example:

- `cheatsheet.local.md`
- `notes.local.md`
- `2026-05-19-notes.local.md`

Files matching `docs/context/*.local.md` are gitignored.
